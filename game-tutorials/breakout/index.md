# Breakout

![Breakout Poster](/game-tutorials/breakout/BreakOut_arcadeflyer.png)

[!badge Easy]

Breakout is a simple game, where you control a paddle and try and destroy a brick wall by bouncing a ball off it. It was [first released in 1976](https://en.wikipedia.org/wiki/Breakout_(video_game)).

There is a great tutorial stepping you through how to build your own version of breakout, which you can find [here](https://developer.mozilla.org/en-US/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript).

You can find the code for the game [here](https://github.com/greytape/breakout).

You can play the game [here](https://greytape.github.io/breakout/).

The tutorial uses Vanilla Javascript, so no frameworks needed.
